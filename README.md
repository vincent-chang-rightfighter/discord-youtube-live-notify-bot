# Discord-youtube-live-notify-bot


## 特色
支援一個 Youtube 頻道直播提醒 <br>
發布至 Discord 單一伺服器 指定 文字頻道<br>


## 環境



## 準備
安裝 Python 3.5 <br>
取得Youtube Data Api<br>
[如何啟動Youtube Api](https://developers.google.com/youtube/v3/getting-started)
## 通知內容格式

預設顯示內容
        
        @everyone
        頻道名稱
        直播主題
        直播網址
